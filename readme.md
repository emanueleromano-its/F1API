Questo progetto consiste nella creazione di una webapp completamente sviluppata in Python utilizzando il framework Flask. L'applicazione consente agli utenti di visualizzare delle statistiche sulla Formula 1 ottenute utilizzando i dati forniti dall'API [F1Open](https://openf1.org/).

## Caratteristiche principali
- Visualizzazione di statistiche dettagliate sui piloti, team e gare di Formula 1.
- Sistema di ricerca per trovare informazioni specifiche.


## Requisiti
- Python 3.7 o superiore
- Flask
- Requests (per effettuare chiamate HTTP all'API F1Open)
